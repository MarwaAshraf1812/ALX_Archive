# 📂 ALX Software Engineering Program Archive

Welcome to my personal archive of the **ALX Software Engineering Program** (offered in partnership with Holberton School). This repository is a comprehensive compilation of all projects, exercises, and tasks completed throughout the curriculum, covering foundational computer science, system engineering, DevOps, front-end, back-end development, and algorithm design.

---

## 🛠️ Tech Stack & Skills Summary

The ALX curriculum spans low-level programming, higher-level scripting languages, system administration, and full-stack web architectures:

- **Languages:** `C` (Low-Level), `Python 3` (OOP, Async, Web), `JavaScript` (ES6+, Node.js), `TypeScript`, `Bash` (Shell Scripting), `SQL`, `HTML5/CSS3`
- **Databases & Caching:** `MySQL`, `PostgreSQL`, `MongoDB` (NoSQL), `Redis` (Caching & Queuing)
- **DevOps & Infrastructure:** `Nginx`, `HAProxy`, `Puppet` (Configuration Management), Docker, SSH, Networking, DNS, Web Server Architecture, Security & Firewalls
- **Core Concepts:** Data Structures & Algorithms, Object-Oriented Programming (OOP), Test-Driven Development (TDD), RESTful APIs, Session & Token Authentication, Background Queues (Bull, Redis)

## 📁 Repository Structure & Curriculum Breakdown

The archive is organized into specialized directories matching each phase of the program:

### 1. Low-Level Programming & Computer Science Foundations
Focuses on deep system-level concepts, memory management, and data structures using the C programming language.
*   **[`alx-low_level_programming`](./alx-low_level_programming)**
    *   *Topics:* Variables, nested loops, pointers, recursion, memory allocation (`malloc`, `free`), static/dynamic libraries, singly/doubly linked lists, hash tables, and binary trees.
*   **[`alx-higher_level_programming`](./alx-higher_level_programming)**
    *   *Topics:* Introduction to Python, object-oriented programming (OOP), file handling, exception handling, test-driven development (TDD), and Python data structures.

### 2. System Engineering & DevOps
Focuses on server administration, automation, script writing, networking, and high-availability web infrastructures.
*   **[`alx-system_engineering-devops`](./alx-system_engineering-devops)**
    *   *Topics:* Shell basics, file permissions, I/O redirections, process management, regex, SSH, web servers (Nginx), load balancing (HAProxy), server monitoring, configuration management with Puppet, and web stack debugging.
*   **[`ALXprodev-Devops`](./ALXprodev-Devops)**
    *   *Topics:* Advanced scripting, DevOps automation patterns, and system optimization.

### 3. Back-End Development & Databases
Covers design patterns, caching, queue management, database administration, and user authentication systems.
*   **[`alx-backend-python`](./alx-backend-python)**
    *   *Topics:* Python variable annotations, asynchronous programming (`asyncio`), decorators, generators, unit/integration testing, and Django applications (including messaging systems).
*   **[`alx-backend-javascript`](./alx-backend-javascript)**
    *   *Topics:* Modern ES6+ JavaScript, promises, classes, TypeScript configurations, Node.js fundamentals, Express.js APIs, and unit testing.
*   **[`alx-backend-storage`](./alx-backend-storage)**
    *   *Topics:* Advanced SQL schema design, query optimization, NoSQL databases (MongoDB), and key-value storage with Redis.
*   **[`alx-backend-user-data`](./alx-backend-user-data)**
    *   *Topics:* User authentication mechanisms (Basic auth, Session auth, JWT tokens, and security measures for personal data).
*   **[`alx-backend`](./alx-backend)**
    *   *Topics:* Pagination design, caching mechanisms, i18n (internationalization), and task-queuing systems using Redis.

### 4. Special Projects & Interview Preparation
*   **[`AirBnB_Clones`](./AirBnB_Clones)**
    *   A multi-phase project replicating the core functionality of the AirBnB platform. Spans from a Python console interface, HTML/CSS frontend, file storage systems, database integrations (MySQL), to a full web dynamic deployment.
*   **[`alx-airbnb-database`](./alx-airbnb-database)** & **[`alx-airbnb-project-documentation`](./alx-airbnb-project-documentation)**
    *   Contains SQL scripts, normalization charts, flow diagrams, and architectural documentation for the Airbnb clone.
*   **[`alx-interview`](./alx-interview)**
    *   Solutions to technical interview questions and algorithms, including: Pascal's Triangle, Lockboxes, Minimum Operations, Log Parsing, UTF-8 Validation, N-Queens Puzzle, Star Wars API integration, 2D Matrix Rotation, and Island Perimeter.
*   **[`alx-pre_course`](./alx-pre_course)** & **[`alx-zero_day`](./alx-zero_day)**
    *   Pre-requisite tasks covering Unix/Linux shell essentials, Git/GitHub flows, and repository setups.

## 🚀 Key Highlights

### The AirBnB Clone Project
The **Airbnb Clone** is a milestone project that ties together the entire full-stack software development lifecycle:
1.  **Console (CLI):** Command-line interpreter to manage backend data models.
2.  **File/Database Engines:** Serialization/Deserialization to JSON and object-relational mapping using SQLAlchemy with MySQL.
3.  **Web Static:** Designing pages with CSS and HTML.
4.  **Web Framework:** Integrating Flask to render dynamic content.
5.  **RESTful API:** Creating endpoints using Flask API.
6.  **Web Dynamic:** Client-side rendering with jQuery.

### Interview Preparation & Algorithms
Solving complex algorithmic challenges (e.g. N-Queens, Lockboxes, log parsers) focusing on time complexity optimization ($O(N)$, $O(\log N)$) and clean, testable code.
